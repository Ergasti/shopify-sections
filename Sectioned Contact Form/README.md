# Sectioned Contact Form

Drag-and-drop contact form builder. Add blocks for email, message, text, textarea, phone, name, headings, checkbox, radio, and select. Fully configurable from the Theme Editor.

**Category:** Forms & Contact  
**Source:** [patrickbolle/shopify-snippets](https://github.com/patrickbolle/shopify-snippets)  
**Last updated in library:** 2025-02-14

## Features

- Block types: Email, Message, Text line, Text area, Telephone, Name, Form heading, Single checkbox, Radio buttons, Select dropdown.
- Reorder blocks in Theme Editor.
- Email and message are required by Shopify; section adds them if not present as blocks.

## Installation

1. Copy `sections/sectioned_contact_form.liquid` to your theme **sections** folder.
2. In Theme Editor, add the “Contact Form” section to the contact page (or any page).
3. Add and arrange blocks, set labels and options.
