# Double Block Section (Double Hero Slider)

Two-block layout: 3/4 width image (left) and 1/4 width image (right). Suited for hero or promotional layouts.

**Category:** Content & Layout  
**Source:** [patrickbolle/shopify-snippets](https://github.com/patrickbolle/shopify-snippets)  
**Last updated in library:** 2025-02-14

## Features

- One “3/4” block and one “1/4” block.
- Image picker and link per block.
- Responsive: stacks on small screens.

## Installation

1. Copy `sections/double-block-section.liquid` to your theme **sections** folder.
2. In Theme Editor, add the section “Double Hero Slider” and set images and links for each block.
