# Videos Slider (Mixed Media Carousel)

Seamless **infinite** carousel of images and videos. Uses cloned first/last slides for smooth looping. No external libraries—vanilla JS + CSS. Supports auto-slide, arrows, visible slides (desktop/mobile), and media ratio (adapt / square / portrait).

**Category:** Video / Media  
**Source:** Custom (SS section)  
**Last updated in library:** 2025-02-14

## Features

- **Two block types:** Carousel Image (image picker) and Carousel Video (video URL). Mix both in one slider.
- **Infinite loop:** Clones leading/trailing slides so the carousel loops without a visible jump.
- **Auto-slide:** Optional with configurable speed (1–9 seconds).
- **Arrows:** Optional prev/next; circle or square; background color.
- **Responsive:** Visible slides (desktop 1–8, mobile 1–8); breakpoint at 767px.
- **Section:** Heading (size, alignment), full width, background color, padding, media size, media design (adapt/square/portrait), optional shadow.

## Installation

1. Copy `sections/mixed-media-carousel.liquid` to your theme **sections** folder.
2. In Theme Editor, add the **Videos Slider** section.
3. Add **Carousel Image** and/or **Carousel Video** blocks; set image or video URL (e.g. from Content > Files for MP4).

## File name

Section file is `mixed-media-carousel.liquid`; the preset name in the Theme Editor is **Videos Slider**.
