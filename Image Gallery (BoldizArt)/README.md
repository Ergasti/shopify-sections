# Image Gallery (BoldizArt)

Dynamic image gallery section. Configurable images per row on mobile, tablet, laptop, desktop; background and text color; container width.

**Category:** Media & Gallery  
**Source:** [BoldizArt/Shopify-Image-Gallery](https://github.com/BoldizArt/Shopify-Image-Gallery)  
**Last updated in library:** 2025-02-14

## Installation

1. Clone or download the [repo](https://github.com/BoldizArt/Shopify-Image-Gallery).
2. Upload files from the **assets** folder to your theme Assets (skip if your theme already has jQuery/Bootstrap).
3. Add a new section in Theme Editor → Sections → Create `image-gallery.liquid` and paste the section code from the repo.
