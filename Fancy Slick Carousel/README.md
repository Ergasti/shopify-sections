# Fancy Slick Carousel Section

Slideshow/carousel section with text overlay, buttons, progress bar, overlay opacity, and multiple slide options. Uses **Slick** slider (jQuery plugin).

**Category:** Carousels & Sliders  
**Source:** [patrickbolle/shopify-snippets](https://github.com/patrickbolle/shopify-snippets) (fancy-slick-carousel.liquid)  
**Last updated in library:** 2025-02-14

## Notes

- Full section code and markup are in the source repo. This library entry is a reference.
- Requires **Slick** (slick-carousel) CSS and JS and jQuery to be included in your theme.
- Copy `fancy-slick-carousel.liquid` from the repo into your theme **sections** folder and ensure Slick is loaded.
