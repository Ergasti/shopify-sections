# Shipping API Tools

Internal tools and API clients for shipping providers used with the Uncovered store.

## Projects

### JT Express MX API
Node.js client for the J&T Express Mexico Open Platform API.
- Order creation, cancellation, and querying
- Logistics tracking and push notification subscriptions
- Zip code coverage checks and freight calculation
- **Stack:** Node.js, HMAC/MD5 signature auth

### JT Express Dashboard
Browser-based dashboard for interacting with the J&T Express MX API.
- Create and track orders visually
- View logistics trajectories
- **Stack:** HTML, CSS (design system), vanilla JS

### ShipBlu Explorer
Browser-based API explorer for the ShipBlu shipping API.
- Interactive interface for ShipBlu endpoints
- **Stack:** HTML, CSS (design system), vanilla JS
