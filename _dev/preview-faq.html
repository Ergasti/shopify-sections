Shopify Sections Library — FAQ Preview

Standalone dev preview of the FAQ section component

A comprehensive, categorized library of Shopify sections and snippets. Copy what you need into your Online Store 2.0 theme — carousels, FAQs, quizzes, video sliders, and more.

Each section has its own folder with a README. Copy the .liquid files to your theme's sections/ and snippets/ folders, and any .js or .css files to assets/. Then add the section via the Theme Customizer.

Carousels & Sliders, Product & Collection widgets, Video sections, Content layouts, FAQ, Quiz, Tooltips, Announcement Bars, Pricing Tables, Maps, and more. Check the main README for the full categorized table.

No. This library has zero build dependencies. The .liquid, .js, and .css files are copy-paste ready. The only tool you might use is Shopify CLI (shopify theme dev) to preview sections in a live development store.
