# Shop by State Map

Interactive India map section. Each state highlights on selection and links to a collection. Configure State Map ID (SVG ID), tooltip label, and collection per state.

**Category:** Maps & Local  
**Source:** [bhoomikakanwarchouhan2104-cpu/shop-by-state-Map-Feature-](https://github.com/bhoomikakanwarchouhan2104-cpu/shop-by-state-Map-Feature-)  
**Last updated in library:** 2025-02-14

## Installation

Copy `State-map.liquid` from the [repo](https://github.com/bhoomikakanwarchouhan2104-cpu/shop-by-state-Map-Feature-) into your theme **sections** folder. Configure state IDs (e.g. INRJ for Rajasthan), tooltips, and collections in the section settings.
