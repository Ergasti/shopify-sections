# SL - Native Video Slider (Section Lab)

Horizontal video slider using **Shopify native video** (upload in Theme Editor). No external URLs—each block uses the theme’s video picker. Scroll-snap, optional autoplay for the active slide, play/pause and mute per slide, optional arrows. Staggered effect and aspect ratio (16:9, 9:16, 1:1) configurable.

**Category:** Video / Media  
**Source:** Section Lab (paid bundle) — **internal use only**  
**Last updated in library:** 2025-02-14

## Features

- **Blocks:** “Video slide” with **Video file** (Shopify native video picker; MP4 from Files).
- **Slider:** Slides per view (desktop 1–6, mobile 1–3), gap, lazy load, autoplay active slide, staggered slides.
- **Slide styling:** Border radius, thickness, color; aspect ratio (16:9, 9:16, 1:1).
- **Controls:** Play/pause and mute on the active slide; optional custom icons (mute, unmute, play, pause).
- **Arrows:** Optional prev/next; show on desktop and/or mobile; size and colors.
- **Section:** Background color, padding, full width or content width.

## Installation

1. Copy `sections/sl-video-slider.liquid` to your theme **sections** folder.
2. In Theme Editor, add the **SL - Native Video Slider** section.
3. Add **Video slide** blocks and use **Video file** to pick a video (upload or select from Files). Only blocks with an MP4 source will show.

## Difference from other video sections

- **Video Slider** (Meetanshi): Uses **video URL** (e.g. CDN link) and Swiper; 3 slides visible.
- **Videos Slider** (Mixed Media): **Image** or **video URL** blocks; infinite loop; auto-slide.
- **SL - Native Video Slider**: **Shopify native video** only (no URL); scroll-snap; play/mute on active slide; no external library.
