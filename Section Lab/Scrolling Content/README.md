# SL Scrolling Content (Section Lab)

Yves Rocher–style **horizontal scrolling marquee**: continuous rolling text with an optional divider between lines. Add blocks for each line; the section duplicates content for seamless infinite scroll. Direction, speed, and typography are configurable.

**Category:** Navigation & Content  
**Source:** Section Lab (paid bundle) — **internal use only**  
**Last updated in library:** 2026-02-24

## Features

- **Text blocks:** Add one block per line; each line can have its own text. Optional divider character (e.g. ✦) between lines.
- **Scroll:** Configurable speed (duration for one full loop, 15–90s), direction (left or right). Animation pauses on hover.
- **Typography:** Font family, weight, responsive font size (min/max), letter spacing. Uppercase, italic.
- **Style:** Text/border color, background, vertical spacing (responsive).

## Installation

1. Copy `sections/sl-scrolling-content.liquid` to your theme **sections** folder.
2. In Theme Editor, add the **SL – Scrolling Content** section.
3. Add **Text line** blocks and set the divider character, speed, direction, and styling as needed.
